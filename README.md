# EDA For Ecommerce Website On Customer Behaviour
---

### Executive Summary:
---
Using Python, SQL and PowerBI 

### Business Problem:
This project explores ecommerce customer behaviour using Python, SQL, and Power BI to uncover insights related to spending patterns, customer segmentation, product performance, and subscription behaviour. The goal is to support data-driven business decisions that improve revenue, retention, and customer experience.

### Methodology:
1. Python(specifically Pandas) to clean and transform data from database and integrate it to MySQL
2. SQL to find meaningful insights in a structured format
3. Build a PowerBI dashbaord to present findings

### Skills:
SQL: CTEs, RANK, Subqueries, Order By, Aggregate functions  
Python: fillNa, groupby, for loops, if statements, maps, feature engineering, data consistency check(redundant data) 
PowerBI:  DAX, donut/bar charts, cards, button slicer
